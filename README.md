# Telegram → ClickUp Bot

Telegram-бот, который принимает задачи в свободной форме, разбирает их через Claude AI и автоматически создаёт задачи в ClickUp с дедлайном, описанием и прикреплёнными файлами.

---

## Что умеет бот

- Пишешь задачу в свободной форме на русском → бот создаёт её в ClickUp
- Автоматически определяет дедлайн («до пятницы», «к 20 марта»)
- Определяет приоритет («срочно», «высокий приоритет»)
- Прикрепляет файлы (PDF, DOCX, PPTX, изображения и любые другие)
- Даёт прямую ссылку на созданную задачу

---

## Шаг 1 — Создать Telegram бота

1. Открой Telegram, найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Придумай имя и username (например `my_clickup_bot`)
4. Скопируй полученный **Bot Token** — он выглядит так: `7123456789:AAHxxxxxx`

---

## Шаг 2 — Получить Anthropic API ключ

1. Зайди на [console.anthropic.com](https://console.anthropic.com/settings/keys)
2. Нажми **Create Key**
3. Скопируй ключ — он выглядит так: `sk-ant-api03-xxxxxx`

> На новом аккаунте есть бесплатный кредит. Стоимость одного разбора задачи — около $0.001.

---

## Шаг 3 — Получить ClickUp API ключ

1. Зайди в ClickUp → кликни на аватар → **Settings**
2. В левом меню найди **Apps**
3. В разделе **API Token** нажми **Generate** или скопируй существующий
4. Ключ выглядит так: `pk_12345678_XXXXXXXXXXXXXX`

---

## Шаг 4 — Найти ClickUp List ID

Это ID списка, куда будут добавляться задачи.

**Способ 1 (через браузер):**
1. Открой нужный список в ClickUp в браузере
2. Посмотри на URL: `https://app.clickup.com/12345678/v/l/li/`**`901234567`**
3. Число после `/li/` — это и есть List ID

**Способ 2 (через API):**
Открой в браузере: `https://api.clickup.com/api/v2/team` с заголовком `Authorization: твой_api_ключ` — там будут все твои пространства и списки.

---

## Шаг 5 — Запуск на Railway (рекомендую, бесплатно)

Railway — облачная платформа, бот будет работать 24/7 без необходимости держать компьютер включённым.

### 5.1 — Загрузи код на GitHub

1. Зайди на [github.com](https://github.com) и создай новый **приватный** репозиторий
2. Загрузи все файлы из этой папки в репозиторий

### 5.2 — Задеплой на Railway

1. Зайди на [railway.app](https://railway.app) → войди через GitHub
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Выбери твой репозиторий
4. Railway автоматически обнаружит `Procfile` и запустит `python bot.py`

### 5.3 — Добавь переменные окружения

1. В Railway открой твой проект → вкладка **Variables**
2. Добавь 4 переменные (нажимай **+ New Variable** для каждой):

| Имя переменной | Значение |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен от @BotFather |
| `ANTHROPIC_API_KEY` | ключ от console.anthropic.com |
| `CLICKUP_API_KEY` | ключ из ClickUp Settings → Apps |
| `CLICKUP_LIST_ID` | ID списка из URL ClickUp |

3. Railway автоматически перезапустит бота — готово!

## Альтернатива: запуск локально (для тестирования)

```bash
# 1. Скопируй .env.example в .env
cp .env.example .env

# 2. Заполни .env своими ключами (открой в любом текстовом редакторе)

# 3. Установи зависимости
pip install -r requirements.txt

# 4. Запусти бота
python bot.py
```

```

├── bot.py              # Основной файл бота

├── clickup_client.py   # Интеграция с ClickUp API
├── config.py           # Загрузка переменных окружения
```
